Android 5.0（API 21）

## 重要（低内存设备注意！！！）

**标签切换策略/播放性能模式**  
设置 → 界面/播放设置 按需设置，首次安装会根据设备 RAM 自动配置

## 功能
- **动态**：视频投稿 / 图文 / 专栏，独立缓存
- **我的内容**：关注 UP 主、收藏、稍后再看
- **播放前详情页**：可选显示视频详情、评论、分享 QR
- **播放器**：播放面板选项可调节
- **开发者模式**：更多可控选项
- **交互**：支持键鼠
