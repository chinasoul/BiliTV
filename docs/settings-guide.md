## 1. 界面设置

### 通用

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **语言** | 下拉选择 | 切换应用显示语言。可选：跟随系统、中文、English。 |
| **省内存模式** | 开关 | 开启后需点击确认键才切换标签页，适合低内存设备。关闭时移动焦点即切换标签。 |
| **启动时自动刷新页面** | 开关 | 开启后每次启动应用时自动刷新首页内容。关闭则使用上次缓存的数据。 |
| **默认启动页面** | 下拉选择 | 设置应用启动后默认显示的页面（如首页、动态、收藏等）。 |
| **显示时间** | 开关 | 开启后在屏幕右上角显示系统时间。 |
| **过渡动画** | 下拉选择 | 页面切换和弹窗等过渡效果。可选：关闭、淡入淡出、左右滑动、缩放淡入。 |

### 主题

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **背景颜色** | 下拉选择 | 设置应用主背景色。可选：纯黑、极深黑、深黑、灰黑、深灰、暗灰、中灰、浅灰、更浅灰等，默认为深灰（#1E1E1E）。 |
| **字体大小** | 下拉选择 | 调整全局字体缩放比例。可选：-30%、-20%、-10%、默认、+10%、+20%、+30%。 |
| **主题模式** | 下拉选择 | 切换深色或浅色主题。可选：深色、浅色。 |
| **主题色** | 色板选择 | 选择应用的强调色/主题色，用于焦点高亮、按钮等元素的着色。提供多种预设颜色。 |

### 布局

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **页面视频列数** | 下拉选择 | 设置一行显示多少个视频卡片。可选：3列、4列、5列、6列。 |
| **播放页右侧边栏宽度** | 下拉选择 | 播放视频时右侧边栏（评论/选集等）占整屏的比例。 |
| **列表边缘渐变** | 开关 | 开启后，滚动列表时上下边缘显示渐变遮罩效果。 |
| **显示滚动条** | 开关 | 开启后在页面右侧显示位置指示条。 |

### 首页分区

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **首页分区开关** | 开关网格 | 控制首页显示哪些内容分区。可切换的分区：推荐、热门、影视、汽车、知识、科技、游戏、动画、舞蹈、美食、生活、音乐、娱乐、鬼畜、时尚、资讯、动物圈、运动。 |
| **首页分区排序** | 排序网格 | 拖拽调整首页各分区的显示顺序。按确认键选中项目，方向键上下移动位置。 |

### 直播分区

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **直播分区开关** | 开关网格 | 控制直播页显示哪些直播分区。 |
| **直播分区排序** | 排序网格 | 拖拽调整直播各分区的显示顺序。 |

### 操作

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **重置本页设置** | 按钮 | 将界面设置页面的所有选项恢复为默认值。 |

## 2. 播放设置

### 播放器

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **播放性能模式** | 下拉选择 | 调整播放器性能策略。可选：高性能（最佳画质体验）、省内存（降低内存占用，适合低配设备）。 |
| **播放完成后** | 下拉选择 | 视频播完后的行为。可选：暂停播放、退出播放、播放下一集、播放推荐视频。 |
| **播放前显示视频详情** | 开关 | 开启后点击视频先展示详情页（简介、选集等），而非直接播放。 |
| **全屏进度条** | 开关 | 开启后在全屏播放时底部显示一条细进度指示条。 |
| **高能进度条** | 开关 | 开启后在进度条上方显示弹幕密度曲线，标识视频中的高能片段。 |
| **隐藏控制栏** | 开关 | 开启后打开视频时不自动显示控制面板，需手动呼出。 |
| **显示播放进度时间** | 开关 | 开启后在播放器左上角显示当前播放时间和总时长。 |

### 解码

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **视频解码器** | 下拉选择 | 设置优先使用的解码器。可选：自动、H.264、H.265、AV1。自动模式下由系统选择最佳解码器。 |
| **默认画质** | 下拉选择 | 每次打开视频时默认请求的画质。可选：360P、480P、720P、1080P、1080P高码率、1080P60帧、4K、HDR、杜比视界、8K。注：高画质需要大会员或登录，未达条件将自动降级。 |
| **默认音质** | 下拉选择 | 每次打开视频时默认请求的音质。可选：64K、132K、192K、杜比全景声、Hi-Res。注：杜比全景声和Hi-Res需大会员，未达条件将自动降级。 |
| **音量均衡** | 下拉选择 | 统一不同视频之间的音量差异。可选：关闭、标准、高动态。 |

### 控制栏按钮

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **控制栏按钮** | 开关网格 | 控制播放器底部控制栏显示哪些按钮。可切换项：播放、评论、选集、UP主、更多、设置、监测、AI配音、循环、详情、关闭、弹幕、互动。至少需保留1个按钮。 |
| **控制栏按钮排序** | 排序网格 | 调整控制栏各按钮的显示顺序。 |

### 操作

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **重置本页设置** | 按钮 | 将播放设置页面的所有选项恢复为默认值。 |

## 3. 弹幕设置

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **开关** | 开关 | 全局开启或关闭弹幕显示。 |
| **不透明度** | 下拉选择 | 调整弹幕的透明程度。可选：10%、20%、30%、40%、50%（默认）、60%、70%、80%、90%、100%。数值越大弹幕越不透明。 |
| **字体大小** | 下拉选择 | 调整弹幕文字大小。可选：极小（12）、小（16）、适中（20，默认）、大（30）、极大（40）。 |
| **占屏比** | 下拉选择 | 弹幕显示区域占屏幕高度的比例。可选值包括 1/8 屏、1/4 屏（默认）、半屏、3/4 屏、全屏。 |
| **速度** | 下拉选择 | 弹幕从右到左滚动的速度。可选：极慢（20秒）、慢（15秒）、适中（10秒，默认）、快（7秒）、极快（4秒）。数值为弹幕穿越整屏的耗时。 |
| **允许顶部悬停弹幕** | 开关 | 开启后显示固定在屏幕顶部的弹幕。关闭则屏蔽顶部固定弹幕。 |
| **允许底部悬停弹幕** | 开关 | 开启后显示固定在屏幕底部的弹幕。关闭则屏蔽底部固定弹幕。 |

### 操作

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **重置本页设置** | 按钮 | 将弹幕设置页面的所有选项恢复为默认值。 |

## 5. 辅助功能（插件）

插件页面顶部显示本机局域网 IP 和二维码，支持两种远程配置方式：
- **方式一**：手机扫描二维码进入配置页面
- **方式二**：在浏览器中输入显示的地址访问配置页面

### 插件列表

每个插件卡片包含：插件名称、描述、版本号、启用/关闭开关。按右键可展开或收起插件的详细设置。

---

### SponsorBlock 插件

自动跳过视频中的广告/赞助片段，片段数据由 bsbsb.top 社区标注提供。

#### 行为设置

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **自动跳过** | 开关 | 开启后自动跳过赞助片段。关闭后将显示手动跳过按钮。 |
| **跳过提示** | 开关 | 自动跳过时是否显示 Toast 提示。 |
| **恰饭统计（调试用）** | 开关 | 在播放器右下角显示当前赞助时间段信息，用于调试。 |

#### 跳过类别

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **类别选择网格** | 开关网格 | 选择要跳过的赞助片段类别（如赞助广告、片头片尾等）。 |

---

### BLana 解析插件

为非大会员提供第三方视频解析能力。仅非大会员用户且开启了 BLana 发现功能时可见。

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **单接口超时** | 下拉选择 | 每个解析接口最多等待多久后切换下一个。可选：3~20 秒。 |
| **解析接口** | 列表管理 | 管理解析接口列表。支持添加、编辑、删除、启用/停用、上移/下移操作。每个接口需填写名称和接口前缀 URL。 |

---

### 弹幕屏蔽插件

根据关键词或正则表达式屏蔽不想看到的弹幕。

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **屏蔽统计（调试用）** | 开关 | 在播放器左下角显示当前被屏蔽的弹幕数量，用于调试。 |
| **包含关键词** | 列表管理 | 弹幕内容包含任一关键词即屏蔽（不区分大小写）。支持添加和删除。 |
| **完全匹配** | 列表管理 | 弹幕内容完全等于关键词才屏蔽（不区分大小写）。支持添加和删除。 |
| **正则表达式** | 列表管理 | 匹配正则表达式的弹幕将被屏蔽。无效正则会显示警告。支持添加和删除。 |

## 6. 本机信息

本页为只读信息展示，显示当前设备的硬件和网络信息。

| 信息项 | 说明 |
|--------|------|
| **信息更新** | 点击刷新按钮更新设备信息（主要是 IP 地址）。 |
| **型号** | 设备型号名称。 |
| **制造商** | 设备制造商名称。 |
| **Android 版本** | 系统版本号及 API 级别。 |
| **CPU 架构** | 设备支持的 CPU 指令集架构。 |
| **GPU** | 图形处理器型号（OpenGL 渲染器名称）。 |
| **设备内存** | 可用内存 / 总内存（单位 MB）。 |
| **屏幕分辨率** | 显示器像素分辨率（宽 × 高）。 |
| **屏幕密度** | DPI 值和密度倍率。 |
| **设计宽度** | 应用 UI 适配使用的设计基准宽度（dp）。 |
| **存储空间** | 可用存储 / 总存储（单位 GB）。 |
| **连接类型** | 当前网络连接方式：WiFi / 有线连接 / 移动数据 / 未连接。 |
| **内网 IP** | 设备在局域网中的 IPv4 地址。 |
| **外网 IP** | 设备的公网 IP 地址（需点击刷新获取）。 |

> 提示：在本机信息页面连续快速点击多次可开启开发者选项。

## 8. 开发者选项

### 播放器调试

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **高能进度条着色** | 开关 | 在高能进度曲线上标记已观看片段。 |
| **播放流程日志** | 开关 | 播放时全屏显示加载、解码、选流过程的详细日志。 |

### 性能监控

| 设置项 | 类型 | 说明 |
|--------|------|------|

| **显示核心频率** | 开关 | 额外显示各 CPU 核心的当前运行频率。 |

### 弹幕渲染

| 设置项 | 类型 | 说明 |
|--------|------|------|

| **视频卡片遮罩透明度** | 下拉选择 | 视频卡片底部渐变遮罩深浅。可选：0.50 ~ 1.00，默认 0.90。 |
| **焦点Alpha** | 下拉选择 | 聚焦状态下主题色背景的深浅。可选：0.20 ~ 0.90，默认 0.60。 |
| **弹窗遮罩透明度** | 下拉选择 | 弹窗弹出时背景变暗的程度。可选：0.30 ~ 0.90，默认 0.60。 |

### 面板/弹窗背景

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **右侧面板背景颜色** | 下拉选择 | 播放器侧栏面板的底色。提供 9 种预设颜色。 |
| **右侧面板背景透明度** | 下拉选择 | 面板底色的透明度。可选：0.30 ~ 1.00，默认 0.70。 |
| **弹窗背景颜色** | 下拉选择 | 弹窗面板的底色。提供 9 种预设颜色。 |
| **弹窗背景透明度** | 下拉选择 | 弹窗面板底色的透明度。可选：0.30 ~ 1.00，默认 0.90。 |
| **弹窗选项文字亮度** | 下拉选择 | 未选中选项的文字白色透明度。可选：30% ~ 100%，默认 70%。 |
| **弹窗选中项加粗** | 开关 | 当前选中的弹窗选项文字加粗显示。 |

### 列表布局

| 设置项 | 类型 | 说明 |
|--------|------|------|
| **滚动减速系数** | 下拉选择 | 列表滚动的减速快慢。越大越快到位，越小越匀速。可选：1.0 ~ 5.0，默认 4.0。 |

| **列表底部间距** | 下拉选择 | 最后一行卡片与屏幕底部的距离。可选：0dp ~ 80dp，默认 4dp。 |
| **滚动条常驻** | 开关 | 关闭后滚动条将在停止滚动后自动隐藏。 |
| **卡片标题行数** | 下拉选择 | 视频/图文/专栏卡片标题的最大行数。可选：1、2、3、4，默认 2。 |
| **卡片标题行间距** | 下拉选择 | 视频/图文/专栏卡片标题的行距倍率。可选：0.8 ~ 1.5，默认 1.0。 |
| **详情页选集最小可用高度** | 下拉选择 | 视频详情页剩余空间不足时选集列表的最小可视高度。默认 220dp。 |
| **正文行间距** | 下拉选择 | 图文/专栏正文描述的行距倍率。可选：0.8 ~ 1.5，默认 1.0。 |
| **评论行间距** | 下拉选择 | 评论内容文字的行距倍率。可选：0.8 ~ 1.5，默认 1.0。 |

|--------|------|------|
| **图片内存缓存限制** | 开关 | 限制 Glide 图片库的内存缓存为 20MB。修改后需重启应用生效。 |

### 动画

| 设置项 | 类型 | 说明 |
