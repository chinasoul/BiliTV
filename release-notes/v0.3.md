## 主要更新内容

### 内存优化 — 修复长时间使用后卡顿/系统重启
- 图片内存缓存从 500张/200MB 降至 100张/50MB，适配 TV 有限内存
- 磁盘缓存从 350 对象/7天 降至 200 对象/3天
- 所有 Tab 刷新时释放旧 FocusNode，防止内存泄漏
- 取消启动时预加载全部 Tab，改为首次切入时按需加载

### 导航性能优化
- 快速按住方向键时使用 jumpTo 替代 animateTo，避免大量动画对象堆积

### APK 签名统一
- 使用固定 release keystore 替代 debug 签名
- 本地构建与 GitHub Actions 构建签名一致，解决覆盖安装失败

### 自动更新改进
- 架构检测改用原生 API 获取设备真实 supportedAbis，准确选择 v7a/v8a

## 升级说明
- ⚠️ 从 v0.2 及更早版本升级需先卸载旧版本（签名变更，仅此一次）
- 老电视32位用v7a，新电视64位用v8a
